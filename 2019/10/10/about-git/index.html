简单记录下自己在Windows简单使用Git的过程

  • 1. 先注册好一个GitHub账号，后面将用到。 GitHub 官网
  • 2. 下载安装git：git 官网。

    不懂就按默认值一路安装下去，最后在开始菜单找到Git Brsh并运行蹦出一个类似命令行窗口的东西，就说明 Git 安装成功！PS：以下代码我都是在Git Brsh窗口输入。

  • 3. 当安装完 Git 应该做的第一件事就是设置用户名称与邮件地址。

    这样做很重要，因为每一个 Git 的提交都会使用这些信息，并且它会写入到每一次提交中，可更改：

    再次强调，如果使用了--global选项，那么该命令只需要运行一次，因为之后无论你在该系统上做任何事情，Git都会使用这些信息。

  • 4. 创建版本库：

    先找个合适的位置，创建空目录确保有权限，我在d盘创建会弹出警告Permission denied，而在c盘不会。

  • 5. 把文件添加到版本库：

    所有的版本控制系统，只能跟踪文本文件的改动，比如TXT文件，网页，所有的程序代码等等，Git也不例外。其他的只能知道大小变化了。千万不要使用Windows自带的记事本编辑任何文本文件。原因是Microsoft开发记事本的团队使用了一个非常弱智的行为来保存UTF-8编码的文件，他们自作聪明地在每个文件开头添加了 0xefbbbf（十六进制）的字符，你会遇到很多不可思议的问题。

    可以add多个文件到缓冲区，再一下子全部commit到仓库

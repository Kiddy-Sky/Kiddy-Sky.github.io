第一步：安装虚拟机

发表于 2021-04-21 更新于 2021-04-26 分类于 OS 阅读次数：
本文字数： 1.2k 阅读时长 ≈ 1 分钟

为了保护自己的电脑，不至于折腾崩了，选择在virtualbox虚拟机中装一个虚拟的Linux系统，然后在 Linux 中装一个Bochs虚拟机来模拟硬件。

1.准备工作
virtualbox可以在官网选择一个合适的版本下载安装。
centos可以在官网选择一个合适的版本下载安装, 若想和书中版本一样的，可以去这里下载安装。我选择镜像下载 CentOS-7-x86_64-DVD-2009.iso 版本的。

2.安装虚拟的 Linux 系统过程
安装 virtualbox 没什么可说的。在 virtualbox 中安装 centos 稍微麻烦了一点。

2.1 在 virtualbox 选择新建

2.2 接着创建

2.3 然后配置
选择存储->选择一个虚拟光盘->选择下载好的CentOS-7-x86_64-DVD-2009.iso

选择系统->将启动顺序的光驱移到第一位。

其他配置

其他配置：共享粘贴板(常规)->网络

->共享文件夹

2.4 在 virtualbox 选择新创建好的虚拟机，点击启动，进入安装界面

注意：Right Ctrl 是指右边的 Ctrl 键。可以切换鼠标在虚拟机外还是虚拟机内。

根据自己的需要选择安装的软件，我安装的是有桌面的。

选择完需要安装的软件后，接着选择安装的位置。如果空间足够，可以直接选择自动创建分区，也可以自己创建分区。我安装的时候空闲空间只有 990MB 左右，根本不够安装桌面版的。所以选择了自己创建分区，然后发现已经有自动创建好的分区了，可能是之前安装的时候创建的，但是当时我安装到重启的时候异常退出了，再启动又是重新安装，所以是原来自动创建的分区。因为原来创建的我也没用过，又不能修改，所以我就直接删掉，然后自己重新创建分区，这样就有足够空闲的空间可以用了。

分区创建好之后点击左上角的完成，会出现如下提示：

点击右下角的 接受更改 即可。

点击 开始安装

2.5 重启配置
重启: 我每次都是重启的时候异常退出，崩了就重新安装，安装了好几次才成功。

3.配置共享文件夹

mkdir /home/QYT/share  # 新建文件夹，用来存放访问共享文件夹的内容
sudo mount -t vboxsf VBshare /home/QYT/share # VBshare：前面设置的共享文件夹的名字，/home/QYT/share 在Linux 中访问共享文件夹内容的文件夹路径

4.安装 Bochs 虚拟机模拟硬件
4.1 下载 Bochs 。我用浏览器自带的下载工具下载太慢了，用迅雷下载就很快。下载后解压到共享文件夹，然后复制到 Linux 系统中，毕竟还是原来的操作系统用得更加方便顺溜。不过要是想要体验用 Linux 终端解压，也不是不行。
我将解压后的 bochs 放进了/home/QYT/OS 中。

4.2 打开终端，切换到 bin 目录，运行
cd /home/QYT/OS/bochs/bin
